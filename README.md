# AI 音樂生成與風格辨認
本專案主要目的為透過HuggingFace 的 MusicGen 模型生成音樂， 並使用CNN 搭配 Transformer 建立的分類模型來判別音樂種類，模型使用的資料集為[GTZAN](https://www.kaggle.com/datasets/andradaolteanu/gtzan-dataset-music-genre-classification)

整合了音樂資料下載、特徵擷取、模型訓練與音樂生成與分類驗證。資料集為 GTZAN，共包含 10 種音樂風格。
使用 CNN 搭配 Transformer 建立分類模型，並透過 HuggingFace 的 MusicGen 模型生成音樂，驗證風格準確性。
